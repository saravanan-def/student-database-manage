# 🎓 Student Database Management System (Python + MySQL)

A simple *CRUD (Create, Read, Update, Delete)* project built using *Python* and *MySQL* to manage student records.  
This project demonstrates database connectivity, SQL queries, and Python programming skills.

---

## ✨ Features
- ➕ Add a new student record  
- 📋 View all student records  
- ✏️ Update existing student details  
- ❌ Delete a student record  
- 💾 Uses MySQL database for storage  

---

## 🛠️ Technologies Used
- Python 3  
- PyMySQL library  
- MySQL database
